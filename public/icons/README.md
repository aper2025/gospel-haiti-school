# App icons

Place the following PNG files here (referenced by `public/manifest.webmanifest`):

- `icon-192.png` — 192×192
- `icon-512.png` — 512×512
- `icon-maskable-512.png` — 512×512 (maskable, safe zone ~80%)

Source image suggestion: `Stroberry-Gospel-Haiti-Logo.jpg` in the team's Downloads folder.
Use a tool like [realfavicongenerator.net](https://realfavicongenerator.net/) or
`@vite-pwa/assets-generator` to generate all sizes from one square master.
