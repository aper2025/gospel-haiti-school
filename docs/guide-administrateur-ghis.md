## 3. Prise de présence (Roland)

### Méthode rapide (sans connexion) :
1. Allez à **https://portal.gospelhaiti.org/take-attendance**
2. Entrez votre **code PIN à 4 chiffres**
3. Choisissez la **classe** à marquer
4. Pour chaque élève, appuyez sur le bouton correspondant :
   - **P** = Présent
   - **LE** = Retard justifié
   - **LU** = Retard non justifié
   - **AE** = Absent justifié
   - **AU** = Absent non justifié
5. Appuyez **« Tous P »** pour marquer tous les élèves présents d'un seul coup
6. Cliquez **« Enregistrer »**
7. Passez à la classe suivante — les classes complétées deviennent vertes

### Depuis le tableau de bord :
1. Cliquez **« Présence »** dans le menu à gauche
2. Sélectionnez la classe et la date
3. Marquez chaque élève
4. Cliquez **« Enregistrer »**

---

## 4. Pointage du personnel (Roland)

### Méthode rapide pour le personnel :
1. Allez à **https://portal.gospelhaiti.org/staff-clock**
2. Chaque membre du personnel clique sur **son nom**
3. Entre son **code PIN à 4 chiffres**
4. Clique **« Arrivée »** le matin ou **« Départ »** le soir

### Suivi dans le tableau de bord :
- Cliquez **« Pointage »** dans le menu
- Vous verrez qui est arrivé, qui est parti, et qui manque
- La section **« Saisie manuelle »** permet de corriger des entrées

---

## 5. Carnet de notes (Roland, Monise, Don Beca)

1. Cliquez **« Carnet de notes »** dans le menu
2. Sélectionnez la **classe**
   - **KG2 à 4F** : toutes les matières s'affichent automatiquement
   - **5F à 9F** : sélectionnez aussi la **matière**
3. Cliquez **« + Ajouter »** pour saisir des notes
4. Choisissez le **type d'évaluation** :
   - Petite évaluation
   - Grande évaluation
   - Dictée
   - Examen
   - Autre
5. Entrez la **date** de l'évaluation
6. Saisissez la **note (0-100)** pour chaque élève
7. Cliquez **« Enregistrer »**

### Groupes automatiques :
Les élèves sont classés automatiquement :
- **G1** (80%+) — En voie de réussite
- **G2** (70-79%) — À surveiller
- **G3** (40-69%) — Intervention requise
- **G4** (<40%) — Crise — tutorat obligatoire

---

## 6. Comportement (Roland, Monise, Don Beca)

1. Cliquez **« Comportement »** dans le menu
2. Cliquez **« + Ajouter »** pour signaler un incident
3. Remplissez :
   - **Élève** : sélectionnez dans la liste
   - **Date** de l'incident
   - **Niveau** (L1 à L6)
   - **Description** de l'incident
   - **Résolution** (optionnel)
4. Cliquez **« Enregistrer »**

### Niveaux de comportement :
| Niveau | Signification | Action |
|--------|--------------|--------|
| L0 | Aucun problème | — |
| L1 | Problème noté | Conversation privée avec l'élève |
| L2 | Répétition | Suivi enseignant |
| L3 | Continu | Contact parent |
| L4 | Sérieux | Administration impliquée |
| L5 | Grave | Rencontre formelle avec parents |
| L6 | Très grave | Directeur impliqué |

**Note :** À partir du niveau L4, l'administration reçoit une notification automatique.

Pour voir les élèves par niveau, cliquez sur les **cases colorées** (L0, L1, L2...) en haut de la page.

---

## 7. Frais et paiements (Roland, Monise, Don Beca)

1. Cliquez **« Frais et paiements »** dans le menu
2. **Barème des frais** : définissez le montant par classe
3. **Enregistrer un paiement** : cliquez **« + Paiement »** à côté du nom de l'élève
4. Remplissez : date, montant (HTG), méthode (espèces, chèque, virement, etc.)
5. Le solde se met à jour automatiquement

### Statuts :
- **Payé en totalité** — vert
- **Paiement partiel** — jaune
- **Impayé** — rouge
- **En retard** — rouge foncé

---

## 8. Personnel (Roland, Monise, Don Beca)

1. Cliquez **« Personnel »** dans le menu
2. **Liste** : voir tous les membres du personnel
3. **Congés** : approuver ou refuser les demandes de congé
4. **Observations** : ajouter des observations de performance
5. Cliquez sur un nom pour voir le profil complet

---

## 9. Horaires

1. Cliquez **« Horaires »** dans le menu
2. **Emploi du temps** : voir l'horaire par classe
3. **Créneaux** : définir les périodes de la journée
4. **Calendrier** : ajouter les jours fériés, semaines d'examens, etc.

---

## 10. Rapports

1. Cliquez **« Rapports »** dans le menu
2. Onglets disponibles :
   - **Quotidien** : absences, retards, pointage, incidents du jour
   - **Hebdomadaire** : données par groupe pour les réunions du mercredi et jeudi
   - **Mensuel** : résumé de présence et paiements du mois
   - **Trimestre / Bulletins** : générer les bulletins scolaires
   - **Relevés** : générer les relevés de notes complets

---

## 11. Élèves

1. Cliquez **« Élèves »** dans le menu
2. Cherchez par nom ou filtrez par classe/statut
3. Cliquez sur un nom pour voir le profil complet
4. Cliquez **« + Ajouter »** pour inscrire un nouvel élève
5. Remplissez toutes les informations (identité, adresse, parents, contact d'urgence)

---

## 12. Suggestions

En bas de chaque page, cliquez **« Suggestions d'amélioration? »** pour envoyer vos idées ou signaler des problèmes. Vos suggestions sont enregistrées et seront examinées.
